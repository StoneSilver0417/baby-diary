# 육아일기 📖

**부부(가족) 전용 비공개 육아일기 앱**입니다.
하루 한 편, 사진과 글로 아이의 하루를 남기고 배우자와 좋아요·댓글로 함께 나눠요.
아이의 성장 기록(키·몸무게·마일스톤)과 주식계좌 투자일기까지 한곳에서 관리할 수 있습니다.

> 앱스토어/플레이스토어 없이 웹 주소로 접속해 홈 화면에 추가하면 일반 앱처럼 쓸 수 있어요.
> 우리 가족만 볼 수 있는 비공개 공간이라, 초대받은 사람만 함께할 수 있습니다.

**주소**: https://baby-diary-tau.vercel.app

---

## ✨ 무엇을 할 수 있나요

- **📔 육아일기** — 하루 한 편, 사진 최대 3장 + 글. 좋아요·댓글로 부부가 소통
- **📅 캘린더 · 🖼 앨범** — 달력으로 그날의 일기 보기, 사진을 날짜별로 모아 감상
- **📈 성장기록** — 키·몸무게를 기록하면 성장 곡선 그래프로, 첫 걸음마 같은 마일스톤도 기록
- **💰 투자일기** — 아이 주식계좌의 매수·매도·배당을 직접 기록하고 평가액 확인
- **👨‍👩‍👧 부부 공유** — 배우자를 초대코드로 초대하면 모든 기록을 함께 보고 남길 수 있어요
- **👶 다자녀 지원** — 아이가 둘 이상이면 아이별로 성장·투자 기록을 따로 관리

---

## 📲 설치하기

설치라고 해도 앱 파일을 받는 게 아니라, **웹 주소를 홈 화면에 추가**하는 방식이에요. 한 번만 해두면 아이콘을 눌러 앱처럼 실행됩니다. 업데이트도 자동으로 반영돼요.

### 아이폰 (Safari)

1. **반드시 Safari로** https://baby-diary-tau.vercel.app 에 접속합니다. (크롬 등 다른 브라우저는 홈 화면 추가가 안 돼요)
2. 화면 하단 가운데 **공유 버튼**(⬆️ 네모에서 화살표가 위로 나가는 아이콘)을 누릅니다.
3. 메뉴를 아래로 내려 **"홈 화면에 추가"**를 누릅니다.
4. 오른쪽 위 **"추가"**를 누르면 끝! 홈 화면에 아이콘이 생깁니다.

### 안드로이드 (Chrome)

1. **Chrome으로** https://baby-diary-tau.vercel.app 에 접속합니다.
2. 화면 아래 **"앱 설치"** 배너를 누릅니다.
   - 배너가 안 보이면 우측 상단 **⋮ 점 3개 메뉴 → "앱 설치"**(또는 "홈 화면에 추가")를 누릅니다.
3. 앱 서랍/홈 화면에 아이콘이 생깁니다.

> 이전에 **APK 파일**로 설치하셨다면: APK 배포는 종료되어 더 이상 업데이트되지 않아요. 기존 앱을 삭제하고 위 방법으로 다시 설치해 주세요.

## 🚀 처음 시작하기

앱을 열면 로그인 화면이 나옵니다.

1. **"계정이 없으신가요? 가입하기"**를 눌러 이메일·비밀번호로 가입합니다. (이메일 인증 절차는 없어요)
2. 가입하면 바로 **온보딩 화면**이 나옵니다. 두 경우 중 하나예요:
   - **내가 가장 먼저 시작** → **"새 가족 만들기"** → 가족 이름, 내 표시 이름(예: 아빠/엄마), 아이 이름·생일 입력
   - **배우자가 먼저 만들고 초대코드를 줌** → **"초대코드로 합류하기"** → 받은 코드와 내 표시 이름 입력
3. 이제 시작이에요! 배우자를 초대하려면 아래 **설정 탭**을 참고하세요.

## 📱 사용법

화면 맨 아래에 **일기 · 성장 · 투자 · 설정** 4개 탭이 있어요.

### 📔 일기 탭

상단의 **피드 / 캘린더 / 앨범** 중에서 볼 방식을 고를 수 있습니다.

**일기 쓰기**
- 오른쪽 아래 **➕ 버튼**을 눌러 작성 화면으로 갑니다.
- 날짜, (아이가 둘 이상이면) 대상 아이, **사진 최대 3장**, 글을 넣고 **저장**.
- **하루에 한 편**만 쓸 수 있어요. 같은 날 다시 들어가면 그날 글이 열려 **수정**됩니다.
- 사진은 여러 장 골라도 자동으로 알맞게 압축돼 저장돼요.

**피드** — 최근 일기가 위에서부터 카드로 보입니다.
- 하트 ❤️ 를 눌러 **좋아요**, 카드를 누르면 상세로 들어가 **댓글**을 남길 수 있어요.
- 댓글은 **내가 쓴 것만 수정(연필)·삭제(×)** 할 수 있습니다.
- 상세 화면 오른쪽 위 연필로 **내 일기 수정**, 휴지통으로 **삭제**.
- 아직 안 본 새 글에는 **"새 글"** 표시가 붙어요.

**캘린더** — 달력에서 일기가 있는 날엔 점이 찍힙니다. 날짜를 누르면 그날 일기로 이동해요.

**앨범** — 사진이 있는 날짜별로 대표 사진이 모여 있어요.
- 여러 장 있는 날은 뒤에 사진이 겹쳐 보이고 장수가 표시됩니다.
- 누르면 그날 사진을 **전체화면으로 좌우로 넘겨가며** 감상할 수 있어요.

**검색** — 상단 돋보기 🔍 로 일기·댓글 내용을 검색합니다.

### 📈 성장 탭

아이의 키·몸무게와 특별한 순간을 기록해요. (아이가 둘 이상이면 위에서 아이를 전환)

- **성장기록** — 키·몸무게를 추가하면 **성장 곡선 그래프**로 그려집니다. 키/몸무게 전환 가능.
- **마일스톤** — "첫 뒤집기", "첫 걸음마"처럼 발달 순간을 날짜와 함께 기록.
- 오른쪽 아래 **➕ 버튼**으로 추가합니다.

### 💰 투자 탭

아이 주식계좌를 직접 기록하는 투자일기예요. (시세 자동 조회는 없고 **수동 기록**입니다. 아이별로 분리)

- **보유 현황** — 종목별 보유 수량·평단가와 평가액이 정리돼 보입니다.
- ➕ 버튼으로 **거래(매수/매도) · 배당 · 메모**를 기록해요.
- 보유 종목을 누르면 **현재가**를 입력할 수 있고, 그 값으로 평가액이 계산됩니다.

### ⚙️ 설정 탭

- **아이 정보** — 이름·생일 수정, **아이 추가**(다자녀).
- **내 표시 이름** — 피드·댓글에 보이는 이름(예: 아빠/엄마)을 바꿔요.
- **가족 구성원** — 함께 쓰는 가족이 보입니다.
- **배우자 초대코드 만들기** — 코드를 발급해 배우자에게 전달하면 온보딩에서 합류할 수 있어요. **72시간 유효, 1회용**입니다.
- **문의하기** — 궁금하거나 불편한 점을 남기면 확인 후 답변드립니다.

## 💡 알아두면 좋은 점

- **업데이트가 안 보일 때**: 앱을 최근 실행 목록에서 **완전히 종료**한 뒤 다시 열어보세요. (이전 화면이 잠깐 캐시될 수 있어요)
- **뒤로 가기**: 어느 화면에서든 뒤로 가면 일기 홈(피드)으로 돌아옵니다.
- **사진**: 하루 최대 3장. 원본은 저장하지 않고 화면에 맞게 압축해 보관해요.
- **비공개**: 초대받은 가족만 접근할 수 있는 우리 가족 전용 공간입니다.
