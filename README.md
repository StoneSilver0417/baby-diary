# 육아일기 설치 안내

부부(가족) 전용 비공개 육아일기 + 아이 주식계좌 투자일기 앱입니다.
하루 사진 3장과 글로 아이의 하루를 기록하고, 배우자와 댓글·좋아요로 소통할 수 있어요.

아래 안내를 따라 설치한 뒤, 회원가입 → 초대코드로 가족 합류(또는 새 가족 만들기) 순서로 시작하시면 됩니다.

---

## 📱 아이폰 설치 (PWA)

아이폰은 앱스토어 설치 없이 "홈 화면에 추가"하는 방식으로 사용합니다. 한 번만 해두면 아이콘을 눌러 일반 앱처럼 실행돼요.

1. **반드시 Safari로** 아래 주소에 접속합니다. (크롬 등 다른 브라우저는 지원하지 않습니다)
   ```
   https://baby-diary-tau.vercel.app
   ```
2. 화면 하단의 **공유 버튼**(사각형에서 위로 화살표 나가는 아이콘)을 누릅니다.
3. 아래로 스크롤해서 **"홈 화면에 추가"**를 선택합니다.
4. 오른쪽 위 **"추가"**를 누르면 완료입니다. 홈 화면에 아이콘이 생깁니다.
5. 이후로는 그 아이콘을 눌러서 실행하면 됩니다.

### 업데이트가 안 보일 때
앱이 이전 버전으로 멈춰 보이면, 홈 화면에서 앱을 위로 스와이프해 **완전히 종료**한 뒤 다시 실행해 보세요.

## 🤖 안드로이드 설치 — 두 가지 방법 중 선택

### 방법 A. 웹앱(PWA) 설치 — 간편, 추천 ⭐

GitHub 로그인이나 별도 설치 파일 없이 바로 시작할 수 있습니다. 업데이트도 자동으로 반영돼요.

1. 안드로이드 휴대폰에서 **Chrome**으로 아래 주소에 접속합니다.
   ```
   https://baby-diary-tau.vercel.app
   ```
2. 화면 아래에 **"앱 설치"** 배너가 뜨면 눌러서 설치합니다.
   - 배너가 안 보이면 우측 상단 **점 3개 메뉴** → **"앱 설치"** 또는 **"홈 화면에 추가"**를 선택합니다.
3. 설치가 끝나면 앱 서랍/홈 화면에 아이콘이 생깁니다. 이후로는 그 아이콘으로 실행하면 됩니다.

이후 업데이트는 그냥 다시 실행하면 자동으로 최신 버전이 반영됩니다. 화면이 이전 버전처럼 이상하게 보이면 앱을 최근 실행 목록에서 완전히 종료 후 다시 열어보세요.

### 방법 B. APK 파일 설치 — 설치형 앱 느낌을 원할 때

GitHub에서 설치 파일(APK)을 내려받아 설치하는 방식입니다. 로그인 없이 바로 받을 수 있습니다.

1. 안드로이드 휴대폰 브라우저(크롬 등)에서 아래 주소에 접속합니다.
   ```
   https://github.com/StoneSilver0417/baby-diary/releases/tag/android-latest
   ```
2. 화면 아래쪽 **Assets**를 펼치고 **`app-debug.apk`**를 눌러 다운로드합니다.
3. 다운로드가 끝나면 알림을 눌러 파일을 엽니다.
4. "출처를 알 수 없는 앱" 관련 경고가 뜨면 **"설정"** → **"이 출처 허용"**을 켠 뒤 뒤로 가서 다시 설치를 진행합니다. (개발 단계 앱이라 뜨는 정상적인 경고입니다)
5. **설치** → **열기**를 누르면 됩니다.
6. 홈 화면에 아이콘이 생기고, 이후엔 그 아이콘으로 바로 실행하면 됩니다.

업데이트가 나오면 같은 링크에서 APK를 다시 받아 그대로 설치(덮어쓰기)하면 됩니다. 데이터는 그대로 유지됩니다.

## 처음 시작하기 (회원가입 · 가족 합류)

1. 앱 실행 후 로그인 화면에서 **"계정이 없으신가요? 가입하기"**를 눌러 이메일·비밀번호로 가입합니다.
2. 가입 직후 온보딩 화면이 나옵니다.
   - **이미 배우자가 먼저 가입해서 초대코드를 받으신 경우** → "초대코드로 합류하기"를 선택하고 코드와 표시 이름을 입력합니다.
   - **가장 먼저 시작하시는 경우** → "새 가족 만들기"를 선택하고 가족 이름·본인 표시 이름·아이 이름·생일을 입력합니다.
3. 배우자를 초대하려면 설정 탭 → "배우자 초대코드 만들기"로 코드를 발급해 전달하세요. (72시간 동안 유효, 1회만 사용 가능)
